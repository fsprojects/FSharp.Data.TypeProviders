FSharp.Data.TypeProviders

The F# Type Providers SqlDataConnection, SqlEntityConnection, ODataService, WsdlService and EdmxFile using .NET Framework generators.

The FSharp.Data.TypeProviders library can be installed from NuGet: 
PM> Install-Package FSharp.Data.TypeProviders

Contents:

  • DbmlFile Type Provider - Provides the types for a database schema encoded in a .dbml file.

  • EdmxFile Type Provider - Provides the types to access a database with the schema in an .edmx file, using a LINQ to Entities mapping.

  • ODataService Type Provider - Provides the types to access an OData service.

  • SqlDataConnection Type Provider - Provides the types to access a SQL database.

  • SqlEntityConnection_ Type Provider - Provides the types to access a database, using a LINQ to Entities mapping.

  • WsdlService Type Provider - Provides the types for a Web Services Description Language (WSDL) web service.

History

This component is shipped in the Visual F# Tools at version 4.3.0.0. The proposal is that subsequent versions and development will happen as an F# community component.
